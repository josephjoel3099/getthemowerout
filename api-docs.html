Check if it's suitable to mow based on weather conditions

API Endpoint

POST /api/check-mowing

Request

Request Body (JSON)

Parameter Type Required Description
location string Yes Town, city name, or postcode (e.g., "London", "Manchester", "SW1A 1AA")

Example Request

curl -X POST https://yourdomain.com/api/check-mowing \
  -H "Content-Type: application/json" \
  -d '{"location": "London"}'

Response Fields

Field Type Description
success boolean Whether the request was successful
canMow boolean True if mowing is advised, False otherwise
location object Location details (name, country, coordinates)
recommendation string Human-readable recommendation
details string Additional context for the recommendation
weather object Current weather data and detailed conditions
timestamp string ISO 8601 timestamp of the response

Deployment

This API is designed to be deployed on serverless platforms:

  • Vercel: Deploy by connecting your GitHub repository to Vercel
  • Netlify: Works with Netlify Functions
  • AWS Lambda: Can be adapted for Lambda with API Gateway

Deploy to Vercel

npm install -g vercel
vercel login
vercel deploy

Rate Limits

This API uses the free Open-Meteo weather service. Please be respectful of their rate limits and consider implementing caching for production use.
